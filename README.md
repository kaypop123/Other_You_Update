# Other-You

> **Other-You**는 두 명의 캐릭터(Adam, Deva)를 전환하며 플레이하는 2D 액션 RPG 프로젝트입니다.  
> 플레이어는 캐릭터를 교체하면서 서로 다른 능력과 전투 스타일을 활용하여 적과 보스를 상대하게 됩니다.  

---

##  프로젝트 개요
- **장르**: 2D 사이드 스크롤 액션 RPG  
- **플랫폼**: PC (Unity 2D 기반)  
- **특징**:
  - 두 캐릭터(Adam, Deva) 전환 시스템
  - 독립적인 체력/에너지/경험치 시스템
  - 스킬 및 쿨다운 UI
  - 보스 AI (FSM 기반)
  - 흡수(Absorption) 및 변신(Transformation) 기믹

---

##  주요 기능
- **캐릭터 전환**: Tab 키로 Adam ↔ Deva 전환, 각각 독립적인 스탯/스킬 보유  
- **전투 시스템**:
  - 근접 공격, 마법 공격, 스킬 사용
  - 데미지 처리, 무적 프레임(i-frames), 피격 이펙트
- **UI 시스템**:
  - 체력/에너지바
  - 스킬 쿨다운 표시
  - 캐릭터별 스탯 패널
- **보스 AI**:
  - FSM(유한 상태 기계) 기반 행동 패턴
  - 거리 기반 공격 선택
  - 랜덤 스킬 시전 및 백대시 회피
- **세이브 시스템**:
  - JSON 기반 슬롯 저장
  - 씬 전환 및 오브젝트 상태 유지

---

##  개발 환경
- **엔진**: Unity (2D 프로젝트)
